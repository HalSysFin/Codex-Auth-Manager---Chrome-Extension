# Security Policy

CAM - Chrome Relay Extension handles browser callback relay data for CAM Auth Manager.

## Supported Use

Use this extension only with a CAM Auth Manager instance you trust.

## Sensitive Data

The extension may handle:

- localhost OAuth callback URLs
- manager base URLs
- bearer tokens configured for manager access

Do not share screenshots, logs, or exported browser storage containing secrets.

## Reporting a Vulnerability

If you discover a security issue, please report it privately to the maintainer before opening a public issue.
